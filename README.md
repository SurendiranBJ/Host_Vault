# Secure Cloud File Manager

A secure web-based file management system built using Flask and MinIO that allows users to upload, download, and manage files with authentication and access control.

## Features
- User registration and login
- Secure file upload and download
- File deletion
- User-based file isolation
- MinIO (S3-compatible) object storage
- Password hashing and session management

## Tech Stack
- Flask
- SQLAlchemy
- PostgreSQL / SQLite
- MinIO
- HTML, Jinja2
